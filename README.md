# 🎵 Spotify Clone

A visually stunning, responsive Spotify web player clone built with modern web technologies. This project replicates the core layout and aesthetic of the Spotify desktop application, featuring a sleek dark mode, interactive cards, and a premium music player interface.

![Spotify Logo](assets/Icons/logo.png)

## ✨ Features

- **🎯 Pixel-Perfect Layout**: High-fidelity reproduction of the Spotify UI.
- **📱 Responsive Design**: Optimized for various screen sizes (Desktop focused).
- **🎨 Premium Aesthetics**: Modern dark theme with Montserrat typography.
- **🔍 Search & Navigation**: Functional-looking search bar and navigation links.
- **🎧 Music Player UI**: Detailed music player bar with progress sliders and playback controls.
- **📚 Library & Playlists**: Sidebar sections for "Your Library", Playlists, and Podcasts.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure for a robust web experience.
- **CSS3**: Advanced styling including Flexbox, Grid, and custom animations.
- **Font Awesome**: High-quality icons for a familiar user experience.
- **Google Fonts**: Montserrat font family for clean, modern typography.

## 🚀 Getting Started

To run this project locally, simply clone the repository and open `index.html` in your favorite browser.

1. **Clone the repository**:


   Navigate to the project folder and open `index.html`.
